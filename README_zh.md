# War3 武侠地图脚手架

一个用于创建魔兽争霸III武侠主题地图的脚手架工程。本项目提供武侠风格的基本素材（地形纹理、模型和地形装饰物），未来还会加入通用的代码框架。

## 项目概述

这是一个完整的武侠主题War3地图开发脚手架，包含：
- 具有武侠美学的基础地形布局
- 中式风格的装饰模型和纹理
- 预配置的装饰物和地形
- 触发器系统设置
- 地图编译工具配置

## 快速开始

### 环境要求
- War3世界编辑器（推荐使用KKWE）
- 基本的War3地图开发知识
- Git版本控制（可选）

### 开发流程
1. **复制scaffold目录** ：将 `scaffold/` 目录复制到您的项目目录中，将目录重命名为您的地图名称。
2. **将地图转换为w3x文件** ：使用w3x2lni工具将您的地图转换为w3x文件。
3. **开始编辑** ：在您喜欢的地图编辑器中打开您的地图文件。您可以使用KKWE（推荐）或其他地图编辑器。

## 资源使用指南

### 地形纹理
地形纹理按主题组织在 `scaffold/resource/terrainart/` 中。为不同地图区域使用合适的纹理：
- `ashenvale/`：森林和神秘区域
- `cityscape/`：城市和建筑区域
- `northrend/`：雪地和山区
- `ruins/`：古代和废弃建筑

### 武侠模型
所有导入的模型都在 `scaffold/resource/war3mapImported/` 中。主要类别：
- **建筑**：寺庙、房屋、墙壁、大门
- **装饰**：灯笼、石头、植被
- **环境**：车辙、废墟、自然元素

### 装饰物配置
装饰物属性在 `scaffold/table/doodad.ini` 中定义。每个条目包含：
- 模型路径引用
- 缩放比例
- 路径属性
- 选择大小和标志
